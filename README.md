# befit

Для добавления/удаления в/из 'Понравилось'/'Не понравилось' необходимо кликать по пальцу вверх/вниз

```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
